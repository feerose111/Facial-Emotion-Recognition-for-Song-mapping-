~EMOTION DETECTION USING CNN FOR SONG MAPPING~

Description

This system incorporates computer vision techniques, including Convolutional Neural Networks (CNNs), For identifying emotions such as Neutral, Happy, sad, and angry by focusing on the dynamic changes in facial muscles. The features extracted from the song align with comparable emotions and are used to suggest songs corresponding to the person's current mood. Leveraging a large musical library, the system selects rule-based method that match the identified emotions.
